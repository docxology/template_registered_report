# manuscript - template_registered_report

Registered-report manuscript for the exemplar. Sections render in filename order:

- `00_abstract.md` — template scope and the bound headline numbers.
- `01_introduction.md` — why the lock-validate-execute-compare ordering matters.
- `02_hypotheses.md` — preregistered hypothesis H1 and the registered outcome
  (embeds `fig:hypothesis_map`).
- `03_methods.md` — frozen registration, analysis plan, and deterministic
  demonstration data (embeds `fig:analysis_dag`).
- `04_results.md` — confirmatory outcome bound to `output/data/demo_analysis.json`
  (embeds `fig:permutation_result`).
- `05_deviations.md` — deviation register and severity classification (embeds
  `fig:deviation_timeline`).
- `06_discussion.md` — scope, limitations, and fork guidance.
- `99_references.md` — bibliography pointer and validation command.

Figures live in [`figures/`](figures/README.md) and are regenerated by
`scripts/generate_figures.py`. Configuration is in
[`config.yaml`](config.yaml); forkable defaults in
[`config.yaml.example`](config.yaml.example). Every reported number is derived
from executed code — do not hand-edit statistics into the prose.
